This is a Serverless Data Pipeline project that uses Terraform for infrastructure orchestration, Python for scripting and deploys to AWS.

## Get Started

### To initialize

```
$ terraform init
```

### To plan

```
$ terraform plan
```

### To deploy

```
$ terraform apply
```
